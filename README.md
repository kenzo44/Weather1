# Weather1

Single Screen Weather Application for NMED 3850

## Features

* Location
* Current Temperature (°C)
* Description of Weather
* Max Temp (°C)
* Min Temp (°C)
* Wind (km/h)
* Humidity (%)

## Extra Features

* Ability to refresh app while in it. Booyeah!
